COVID19 API v1 Documentation

This is the COVID19 API, which utilizes the third-party API from TianAPI, WAPI, and Juhe

Important details:

  • No need to use CORS proxy or API key
  • Only GET requests are allowed

GET

Returns the data from a specific endpoints

Path: https://covid-19-api-server.herokuapp.com

Routes

Path Required Description
/api Yes Fetches China Data from Tianapi
/api/abroad Yes Fetches World Data from Tianapi
/api/china Yes Fetches China Data from Wapi endpoint
/api/world Yes Fetches World Data from Wapi endpoint
/api/travel No Fetches Travel city from Juhe. You only need to run it once. The data will remain the same

Example URL: https://covid-19-api-server.herokuapp.com/api
